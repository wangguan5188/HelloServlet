Struts2框架 使用环境搭建
一、下载Struts2框架源代码
	地址：http://struts.apache.org/
	
二、使用步骤：
1、导入jar包
	
	新建一个web工程，将Struts2源代码的apps目录下的struts2-blank。war文件解压缩，
	找到解压缩的WEB-INF目录下的lib文件夹，将其里面所有的jar包复制到新建的web工程的lib文件夹下。
	
2、在web.xml文件中配置Struts2的前端控制器。
	复制 struts2源代码\apps\struts2-blank1\WEB-INF\web.xml 文件中的过滤器的配置到
	当前 web 应用的 web.xml 文件中。或增加如下配置：
	
	<filter>
		<filter-name>Struts2</filter-name>
		<filter-class>org.apache.struts2.dispatcher.ng.filter.StrutsPrepareAndExecuteFilter</filter-class>
	</filter>
	<filter-mapping>
		<filter-name>Struts2</filter-name>
		<url-pattern>/*</url-pattern>
	</filter-mapping>

3、在项目的src目录下新增struts.xml文件并关联Struts2的DTD文件（关联之后可以实现代码自动提示）。

 	复制 struts2源代码\apps\struts2-blank\WEB-INF\classes 下的 struts.xml 文件到
 	当前 web 应用的 src 目录下。

	关联Struts2的DTD文件：工具栏--windows--Preferences--XML--XML Catalog--add
		location:file system...  --找到struts2源代码\src\core\src\main\resources\struts-2.3.dtd
		Key Type:URI
		Key:http://struts.apache.org/dtds/struts-2.3.dtd
		
4、在struts.xml配置请求与action的对应关系，在action中处理业务逻辑。
